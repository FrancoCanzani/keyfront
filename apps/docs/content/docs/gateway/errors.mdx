---
title: Errors
description: HTTP errors returned by the Keyfront gateway.
---

| Status | Meaning |
| --- | --- |
| `401` | The API key is missing, invalid, or revoked. |
| `402` | The publisher account cannot currently accept traffic. |
| `429` | The key exceeded its plan limit. |
| `502` | Keyfront could not obtain a valid response from the origin. |

This contract will be expanded with stable error codes before public launch.
